Descripcion del proyecto
El proyecto consiste en dar una solucion a la problematica de cargar archivos de centros de cultivo con sus elementos como nombre, comuna y toneladas con el objetivo de identificar al centro con mayor produccion

Estructura general del proyecto
El proyecto esta estructurado en 3 paquetes Model, data y UI en la que la clase Main esta contenida en paquete UI, meintras que la clase data contiene el gestor de datos y el paquete model contiene la clase para los centros de cultivos
